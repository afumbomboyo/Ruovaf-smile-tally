# **App Name**: Ruovaf Smile Tally

## Core Features:

- Personal Smile Counter: A dedicated button for each partner to increment their personal 'smiles for you' count with a single click.
- Shared Smile Display: A dashboard view showing both partners' current smile counts, but only if the respective partner has enabled visibility for their count. This visibility can be toggled on and off by each partner.
- Partner Identification: Simple authentication or selection mechanism to ensure each partner can only increment their own count.
- Persistent Smile Data: Secure storage of each partner's smile counts so they are preserved across sessions. Daily counts refresh, but all historical daily records are also kept.
- AI-Generated Loving Prompts: After a partner clicks to record a smile, an AI tool provides a short, sweet prompt or thought that encourages reflection or sharing of affection.

## Style Guidelines:

- Primary color: A warm, vibrant rosy pink (#EE4D93), conveying affection and joy. This color will be used for interactive elements and highlights.
- Background color: A very light, subtle pink (#F8F0F3), creating a soft and comforting canvas that allows the primary elements to stand out.
- Accent color: A bright, soft coral-red (#FE6E6E), used sparingly for impactful actions or to draw attention to new milestones.
- Body and headline font: 'PT Sans' (humanist sans-serif), chosen for its modern aesthetic combined with a warm, friendly personality, ensuring high readability.
- Use simple, clean line-art icons that evoke themes of love, smiles, and hearts, complementing the app's straightforward functionality.
- A clean, minimalist layout with prominent, easy-to-tap buttons and clear, centered display of smile counts for immediate understanding.
- Subtle, smooth animations for button presses and count increments, providing satisfying visual feedback without distracting from the core interaction.
